# Coda.io Feature Research

**Date:** 2026-01-15 11:05
**Purpose:** Feature inspiration for Field Ops

---

## Core Platform

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **Docs + Tables** | Unified docs with embedded interactive tables | Work order notes + task tables in one view |
| **Building Blocks** | Drag-and-drop components (headings, callouts, tables) | Modular work order templates |
| **Views** | Same data, multiple views (table, board, calendar, Gantt, grid) | WO list → calendar → map → timeline |

## Automation

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **When/Then Rules** | Time-based, row-changed, form-submitted triggers | Auto-assign tech when WO created |
| **Webhook Triggers** | External events trigger actions | Customer portal triggers dispatch |
| **Conditional Logic** | If X then Y automation chains | SLA escalation workflows |

## Collaboration

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **Cross-doc Sync** | Tables sync across documents | Shared customer data across WOs |
| **Comments & Mentions** | @mentions in any cell/row | Tag techs on specific tasks |
| **Revision History** | Track all changes | WO audit trail |
| **Role-based Access** | View/edit/play permissions | Tech vs dispatcher vs admin views |

## Data & Tracking

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **Sub-items/Subtables** | Nested rows within rows | Tasks within work orders |
| **Timeline Dependencies** | Task dependencies visualization | Job scheduling with dependencies |
| **Custom Columns** | Text, dates, checkboxes, assignees, files | Flexible WO fields |
| **Filters & Grouping** | Dynamic data slicing | Filter WOs by tech, status, date |

## Integrations (Packs)

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **7000+ via Zapier** | Connect any app | QuickBooks, Salesforce sync |
| **Native Packs** | Slack, Gmail, Jira, GitHub | Push notifications, email triggers |
| **API Access** | Programmatic control | Mobile app sync, external systems |

## AI Features

| Feature | Description | Field Ops Inspiration |
|---------|-------------|----------------------|
| **AI Content Generation** | Auto-write descriptions | Generate WO summaries from notes |
| **AI Insights** | Analyze data patterns | Predict busy periods, tech utilization |
| **Task Delegation** | AI handles repetitive work | Auto-categorize incoming requests |

## Templates

Pre-built for: project management, meetings, OKRs, knowledge bases, product roadmaps

**For Field Ops:** Could offer WO templates by trade type (electrical, HVAC, plumbing)

---

## Top 5 Features Worth Stealing

1. **Multiple views of same data** (table/board/calendar/timeline)
2. **When/Then automation rules** (visual, no-code)
3. **Sub-items** (tasks nested in WOs)
4. **Cross-doc sync** (shared customer data)
5. **AI summaries** (auto-generate WO reports)
